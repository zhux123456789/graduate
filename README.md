##🛠 功能说明

###1.Scrapy爬虫（`豆瓣/`)
-完整的Scrapy框架实现
-支持自动翻页和数据持久化
-需配置代理IP(见注意事项)

### 2. Requests 系列
| 目录                | 特性                          |
|---------------------|-------------------------------|
| `request-model`     | 基础 Requests 实现             |
| `request-ip-agent`  | 增加代理IP池和多线程优化       |

### 3. Selenium 系列
| 目录                | 特性                          |
|---------------------|-------------------------------|
| `selenium-model`    | 基础浏览器自动化操作           |
| `selenium-behavior` | 模拟人类行为（随机滚动/停顿等）|

###4.数据输出
-所有爬虫结果均保存为`csv`文件
-字段包含：爬取的数据和清洗后的数据

##⚠️ 重要注意事项

###浏览器驱动
1. `msedgedriver.exe`必须与本地**边缘浏览器版本匹配**
   -不匹配会导致运行失败
   -更新浏览器后需重新下载对应驱动

###反爬策略
1. **饼干维护**（一定要及时更换有效Cookie)
   -登录后获取的饼干可爬取全部页数
   -未登录仅能爬取前5页数据
    

2. **用户代理**
   -若被拦截,需更换`用户代理`列表中的值

3. **代理互联网协议(Internet Protocol)**
   -当前使用付费代理IP(有效期至 2025年6月)
   -剩余IP:100+个
   - **到期或耗尽后需手动更换**代理池配置
